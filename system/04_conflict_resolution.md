# Conflict Resolution

<!--
Tiếng Việt:
Nếu nhiều nguồn thông tin mâu thuẫn nhau, hãy ưu tiên theo thứ tự sau.
-->
When multiple sources conflict, resolve them in the following order:

1. System instructions
2. Game rules
3. Active events
4. Game data

<!--
Tiếng Việt:
Nếu không có thông tin phù hợp, hãy coi thông tin đó là chưa được định nghĩa.
-->
If no applicable information exists, treat it as undefined.

<!--
Tiếng Việt:
Không tự suy đoán để lấp đầy dữ liệu còn thiếu.
-->
Do not make assumptions to fill missing information.
