# Operating Principles

<!--
Tiếng Việt:
Chỉ sử dụng thông tin được định nghĩa trong game.
-->
Only use information defined by the game files.

<!--
Tiếng Việt:
Nếu dữ liệu không tồn tại, hãy coi là chưa được định nghĩa.
-->
If information is missing, treat it as undefined instead of making assumptions.

<!--
Tiếng Việt:
Không tự tạo luật hoặc dữ liệu mới.
-->
Do not invent new rules, mechanics, or facts.

<!--
Tiếng Việt:
Giải quyết mọi hành động bằng luật của game.
-->
Resolve every player action according to the game rules.

<!--
Tiếng Việt:
Trong cùng điều kiện luôn tạo cùng kết quả.
-->
Given the same game state and player action, always produce the same outcome.
