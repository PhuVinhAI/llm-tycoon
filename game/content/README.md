# Content

## Purpose

This directory contains the game data of LLM Tycoon: the concrete catalogs, tables, and calendars that the Rules operate on.

Content follows the mechanics defined in the Rules and never introduces new mechanics. Growing the game (more technologies, more contracts, more events) usually means editing only this directory.

The data is grounded in real AI history (2013–2020): papers, hardware releases, datasets, and market waves appear roughly when they did in reality.

---

# Scope

* Technology tree and architecture specifications.
* Tasks, match matrix, and domain fit lists.
* Market demand by era.
* Hardware shop and dataset market.
* Contract pool and employee candidates.
* The Event Calendar and competitions.
* Score titles.

---

# Ordering

Files are numbered in reading order. The build assembles them in file-name order.

## Tiếng Việt

Thư mục này chứa dữ liệu game: các danh mục, bảng số liệu và lịch sự kiện mà
Rules vận hành lên. Content tuân theo cơ chế trong Rules và không tạo cơ chế
mới — muốn mở rộng game (thêm công nghệ, hợp đồng, sự kiện) thường chỉ cần
sửa thư mục này.

Dữ liệu bám theo lịch sử AI thật (2013–2020): các bài báo, đợt ra mắt phần
cứng, bộ dữ liệu và làn sóng thị trường xuất hiện gần đúng thời điểm ngoài
đời.
